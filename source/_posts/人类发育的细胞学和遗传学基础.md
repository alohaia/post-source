## 基因的基本结构

基因
: 具有功能的 DNA 片段

{% note warning%}
注意基因并不仅仅包含**编码DNA**
{% endnote %}

断裂基因
: DNA 片段由**编码序列**和**非编码序列**组成，非编码序列将编码序列隔开

人类基因
- [外显子](#外显子)
- [内含子](#内含子)
- [侧翼序列](#侧翼序列)

### 外显子

外显子（exon）
: 基因内的编码序列

### 内含子

内含子（intron）
: 基因内的非编码序列

GT-AG 法则
: 每个每个内含子的 5' 端开始的两个核苷酸都是 GT，3' 端末尾的两个核苷酸都是 AG
: 是 RNA **剪接**的信号

### 侧翼序列

侧翼序列（flanking sequence）
: 每个基因的 5' 端和 3' 端都有的一段不转录的非编码区

#### 启动子

启动子（promoter）
: 位于转录起始点上游 100~200bp 内，为**RNA 聚合酶**结合部位，启动基因转录

启动子的几种序列：

TATA 框（TATA box）
: 位于转录起始点上游 25~30bp，由 TAT(A/T)A(A/T)A 七个碱基组成，与转录因子 TFII 结合，精确定位转录起始

GAAT 框（GAAT box）
: 位于转录起始点上游 75~80bp，由 GG(C/T)CAATCT 九个碱基组成，与转录因子 CTF 结合，高效转录所必须

GC 框（GC box）
: 位于 CAAT 框的两侧，序列为 GGGCGG，与转录因子 SP1 结合激活转录

#### 增强子

增强子（enhancer）
: 一般包括**启动子**上、下游的一段 DNA 序列，能显著提告基因的转录效率
: 不受距离变化的影响，作用方向可以是 5'→3'，也可以是 3'→5'
: 具有*组织特异性*，与基因的选择性表达有关

## 基因的表达与调控

### 基因表达

基因表达
: 即基因的**转录**和**翻译**

==mRNA 的成熟加工==：mRNA 前体（核内异质 RNA | hnRNA）加工成有功能的成熟 mRNA

- **剪接**
- **加尾**/**多聚腺苷酸化**

- 转录前调控
- [转录调控](#转录调控)
- [转录后调控](#转录后调控)
- 翻译调控
- 翻译后调控

#### 转录调控

转录调控
: 主要通过**顺式作用元件**和**转录因子**之间的相互作用来实现

转录因子 | 反式作用元件
: 能与目的基因调控序列中*顺式作用元件*相结合调节基因表达的**蛋白质**
: 有*激活因子*（activator）、*抑制因子*（repressor）

顺式作用元件
: 有[启动子](#启动子)、[增强子](#增强子)、[终止子](#终止子)、沉默子

沉默子 | 绝缘子
: 一般位于上游 20~2000bp 不等，某些沉默子可以位于启动子的下游、基因本身的内含子或外显子上。

#### 转录后调控

**RNA 剪接 | 选择性剪接**

RNA 编辑
: 少见于哺乳动物
: 如人载脂蛋白基因 APOB 在肝脏中编码 apoB100，在长中编码 apoB48

#### 表观遗传调控

表观遗传调控
: 有丝分裂和/或减数分裂中*可遗传*的基因功能改变，*不涉及 DNA 序列的改变*

1. DNA 甲基化
2. 组蛋白修饰
3. 染色质重塑
4. 非编码 RNA

DNA 甲基化
: **CpG** 二核苷酸的**胞嘧啶**（C）共价结合一个甲基基团，一般会**抑制转录**

组蛋白修饰
: **乙酰化**（激活转录）、*甲基化*、磷酸化、泛素化

染色质重塑
: 所有能使染色质发生功能性变化的染色质修饰和重构

- 组蛋白共价修饰
- DNA 分子甲基化
- RNA 干扰
- ATP 依赖的染色质重塑：由 **ATP 依赖的染色质重塑复合物**改变组蛋白和
  DNA 的结合状态，使转录因子易于接近 DNA

**非编码 RNA**
- 小干扰 RNA（small interfering RNA，siRNA）：*外源或内源性双链 RNA*，长度 21~23nt
- 微小 RNA（microRNA，miRNA）：*内源性非编码单链 RNA*，长度 18~25nt
- 长链非编码 RNA（long noncoding RNA，lncRNA）：无编码蛋白潜能，长度大于 200nt

## 基因突变

突变（mutation）
: 遗传物质的可遗传变异

- **染色体畸形**：染色体数目或结构异常
- **基因突变**：DNA 变化
    1. 点突变
    2. 移码突变
    3. **动态突变**

点突变 | 单碱基突变 | 单碱基置换
: 即单个碱基置换，指一个碱基被另一个碱基所替换

- 错义突变
- 同义突变 | 中性突变 | 静止突变
- 无义突变：编码氨基酸的密码子突变为终止密码子
- 终止密码子突变：终止密码子突变为编码氨基酸的密码子

移码突变
: DNA 编码序列插入或缺失 3 的非整数倍个碱基
: 导致阅读框架改变，结果突变点以后的氨基酸种类和顺序改变

动态突变
: **三核苷酸**重复（triplet repeat）次数在世代传递过程中明显增加
: 形成原因是三核苷酸重复序列在复制时发生位移

DNA 多态性
: 某一**基因座**在人群中存在**两种或两种以上**的序列突变，其中频率最低的形式也远远高于依赖突变所能维持的频率
: 对于同一基因座上的两个或两个以上的等位基因，等位基因频率至少 **0.01** 携带该等位基因的杂合子频率大于 **2%**，则*认为该基因座具有多态性*

- RFLP（限制性片段长度多态性）
- VNTR（可变数目串联重复）
- SNP（单核苷酸多态性）

## 基因分析技术

鉴定单基因突变的流程
1. 临床：诊断→系谱绘制→查阅文献→采血

### 分子杂交

分子杂交
: 利用碱基互补碱基配对的原理，在复杂的靶 DNA 或 RHA 序列中，用特异的探针来鉴定其同源的 DNA 或 RNA 片段

1. Southern 印记杂交
2. Northern 印记杂交
3. *Western 印记杂交*
4. 斑点杂交

#### 探针

探针（probe）
: 一段与目的基因或 DNA/RNA 片段特异杂交的核苷酸序列

探针种类
- DNA 探针
- RNA 探针
- 寡核苷酸探针

探针的标记
- 切口平移法
- 随机引物延伸法
- DNA 末段标记法
- RNA 探针标记

探针检测
- 放射性同位素
- 荧光基因
- 间接检测

#### 杂交技术

Southern 印记杂交
: 通过标记探针的 DNA 单链与互补的待测靶 DNA 单链结合形成异源双链，检测待测靶 DNA 序列中是否存在与探针同源的 **DNA 序列**
: 广泛应用于遗传学研究和基因诊断

Northern 印记杂交
: 通过标记探针的 DNA 单链与互补的待测靶 mRNA 单链结合形成异源双链，检测待测靶 mRNA 序列中是否存在与探针同源的 **mRNA 序列**
: 主要应用于检测某一基因的 mRNA 表达水平和相关转录物的情况

Western 印记杂交 | 免疫印记
: 通过运用免疫动物或遗传工程方法生产的抗体来检测待测样品是否有能够与抗体特异性结合的抗原（**蛋白质**）
: 主要应用于基因表达和功能研究

斑点杂交
: （一般）使用等位基因特异性寡核苷酸（allele-specific oligonucleotide, ASO）探针，快速筛查**等位基因间单核苷酸的差异**

### 重组 DNA 技术

重组 DNA 技术
: 遗传工程的**核心技术**
: 按人类需要选择*目的基因*（DNA 片段），在体外与*载体*重组，再用一定的基因转移方法将重组 DNA 输入另一细胞或生物体内，以改良或创新生物品种以及治疗人类疾病

所需工具、技术
1. 限制酶
2. 载体
3. 细胞 DNA 克隆
4. 基因文库

#### 限制酶

限制酶 | 限制性内切核酸酶
: 能特异性切断 DNA 链内的磷酸二酯键

**II 型限制酶**的酶切位点在其识别序列的内部，可以应用于重组 DNA 技术

限制酶的命名
> 如 *Alu* I 是分离自属和种名为 Arthrobacter luteus 细菌的第一种酶
- 细菌属名第一个字母（大写斜体）
- 细菌种名的前两个字母（小写斜体）
- *菌株名称（大写或小写正体）*
- 酶的序号（罗马数字）

#### 载体

载体（vector）
: 将外源 DNA 片段运送到受体细胞并能进行自我复制增殖和表达的工具

- 克隆载体：外源 DNA 片段，增殖扩大数量
- 表达载体：外源基因，产生蛋白质

**载体的特征**
1. *分子质量小*，便于与较大的 DNA 片段结合，能进入受体细胞并在其中增殖
2. 有多个*限制酶切位点*，但对每一种限制酶只能有一个酶切位点
3. 插入外源 DNA 后，不影响自身的*复制*和*表达*能力
4. 有可选择的*标记基因*（如抗药基因）或报道基因
5. 克隆载体上要有*复制起点*，表达载体上**还**要有*启动子*

常用载体
- <a href="{% post_path 细菌%}#细胞质">质粒</a>：能克隆较小 DNA 片段（小于 10kb）
- <a href="{% post_path 病毒%}#噬菌体">λ 噬菌体</a>：能克隆中等长度的 DNA 片段，具有 **<a href="{% post_path 病毒%}#噬菌体">cos 序列</a>**
- 黏粒
- 细菌人工染色体：可克隆长达 300kb 的 DNA 大片段
- 酵母菌人工染色体：常用来克隆很大的 DNA 片段

黏粒（cosmid）
: 将质粒和噬菌体改建的一种载体 , 即将 λ 噬萧体的 cos 序列插入质粒

细菌人工染色体（BAC）
: 一种重组的质粒，能在细菌细胞内繁殖

酵母菌人工染色体（YAC）
: 一种在酵母中繁殖的克隆载体系统

#### 细胞 DNA 克隆

细胞 DNA 克隆 | DNA 克隆 | 分子克隆
: 为获得特异基因和特异序列，将目的基因 DNA 片段与载体 DNA 重组，使重组 DNA 分子在**宿主细胞**中表达扩增而获得大量相同的 DNA 片段

{% asset_img 细胞克隆的选择增殖.png "细胞克隆的选择增殖'细胞克隆的选择增殖'" %}

步骤
1. 构建重组 DNA 分子：将靶（目的）DNA 片段与载体连接
2. 转化：将重组 DNA 分子转移到宿主细胞（细菌或酵母细胞）
3. 细胞克隆的选择增殖：将转化的细胞值置于*琼脂皿的表面*以刺激其克隆生长，再将单个**细胞克隆**移到*液体培养基*中进行扩增
4. 分离重组 DNA 克隆：收获扩增的培养细胞并选择分离重组 DNA

细胞克隆
: 单个细胞在遗传上形成的在遗传上相通的细胞群体

- 基因组 DNA 文库

### PCR

PCR | 聚合酶链反应 | 多聚酶链式反应
: 一种体外 DNA 扩增技术，模拟体内 DNA 复制条件，可以在体外迅速地选择、扩增特定的靶 DNA 序列，得到大量相同的 DNA 片段，即 **PCR-DNA 克隆**

设计原理及步骤
1. 设计及合成引物（primer）
2. 必须有 DNA 前身物（四种 dNTP）
3. 必须有 *Taq* 酶
4. PCR 反应体系：基因组 DNA、一对引物、四种 dNTP、*Taq* DNA 聚合酶、酶反应缓冲体系、必需的离子（如 Mg^2+^）
5. PCR 是链式反应：变性→复性→延伸→变性→……
